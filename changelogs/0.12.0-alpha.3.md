# Lunil 0.12.0-alpha.3

Lunil 0.12.0-alpha.3 adds typed syntax projections for embedded source-analysis tools while
preserving the existing lossless syntax tree.

## Typed syntax facades

- Added typed projections for normal and method calls, dot and colon member access, function
  declarations and expressions, qualified names, parameters, and function blocks.
- Added constant UTF-8 string and identifier-token helpers for common call-site extraction.
- Added explicit nullable recovery slots and `IsComplete` checks. The underlying `LuaSyntaxNode`
  remains available for malformed-source handling and advanced lossless inspection.

## Visitors and workspace integration

- Added extensible `LuaSyntaxWalker` and `LuaSyntaxVisitor<TResult>` dispatch over the typed views.
- Updated workspace dependency discovery to use the public call facade and walker rather than
  depending on parser child order.
- Parenthesized calls, shorthand string and table calls, method calls, named/local/global/anonymous
  functions, and malformed recovery shapes are covered by the typed contract.
