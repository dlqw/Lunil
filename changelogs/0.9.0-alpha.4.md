# Lunil 0.9.0-alpha.4

## Calls and floating-point regions

- Added compiled paths for stable fixed-shape primitive Lua leaf calls with exact numeric and
  Boolean argument and result types.
- Reused closure guards within a compiled invocation while retaining generation, hook, finalizer,
  and target-mismatch fallbacks.
- Preserved scheduler fallback for varargs, open results, upvalues, recursive or native calls,
  debug hooks, yields, budget exhaustion, and invalidation.
- Updated JIT profile data for fixed call argument kinds.

## Performance

- Improved function-call and floating-point control-flow workloads while preserving the existing
  limits for calls, generated code, registers, instructions, and backedges.
