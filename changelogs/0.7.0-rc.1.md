# Lunil 0.7.0-rc.1

## Release candidate promotion

- Promote the accepted `0.7.0-beta.1` product to its first release candidate. This promotion makes
  no product-code change and does not alter any of the 14 frozen public API baselines or the
  version-independent 14-package scope.
- Close compatibility, diagnostics, documentation, reliability, and performance hardening for
  normal RC development. `0.7.0-rc.1` accepts only release-blocking fixes; new features and API
  redesigns move to a later numeric milestone.
- Keep the official Lua 5.4.8 user-mode suite, six-backend observable goldens, deterministic fuzz,
  GC/coroutine soak, NativeAOT, trimming, ReadyToRun, package-consumer, and six-RID bundle contracts
  unchanged from the accepted Beta.

## Accepted release evidence

- The `0.7.0-beta.1` accepted commit passed main-branch six-RID CI in run `29371180602`.
- Its immutable `v0.7.0-beta.1` tag passed the clean tag-triggered release workflow in run
  `29372551793`, producing all six RID bundles, 14 NuGet packages, 14 symbol packages, clean local
  consumers, installed CLI tool smoke, compatibility validation, package publication, and the
  GitHub prerelease.
- RC validation re-runs the same version-derived gates against `0.7.0-rc.1`; an accepted candidate
  is promoted to stable `0.7.0` without product-code changes.
