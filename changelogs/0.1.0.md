# 0.1.0

## Added

- 确立 Lua 5.4.8、.NET 10、解释器、CIL JIT、CIL AOT 与 NativeAOT 的总体架构。
- 确立 PUC Lua 5.4 binary chunk 双向兼容目标。
- 确立默认 LuaLS、可选旧 EmmyLua IDE 方言的语义分析策略。
- 建立编译器解决方案及 byte-oriented source、chunk codec/verifier 的初始实现。
- 实现完整 Lua 5.4 token/trivia 词法模型、资源预算和稳定诊断。
- 实现 Lua 短字符串、长字符串、转义和数值字面量的 PUC 兼容解码。
- 实现覆盖全部 Lua 5.4 语法的保真、容错语法树解析器。
- 添加表达式优先级、右结合、错误恢复、lossless token 和恶意输入测试。
- 实现局部变量、参数、upvalue、`_ENV`、只读属性和隐式 `self` 的语义绑定。
- 实现 vararg、break、label/goto 可见性和禁止跳入局部变量作用域的检查。
- 添加多层闭包捕获传播、repeat 作用域及 PUC 静态语义差分测试。
- 新增版本化 canonical register IR、显式基本块、指令副作用和独立 IR verifier。
- 实现从无错误 CST/绑定模型到 canonical IR 的 lowering，覆盖求值顺序、多返回值、短路、表、函数捕获、goto、作用域关闭及全部循环形式。
- 新增 16-byte `LuaValue`、二进制 `LuaString`、array/hash `LuaTable`、显式 Lua 栈/帧、闭包和 open/closed upvalue cell。
- 新增基线 canonical IR 解释器，支持 Lua/宿主调用、vararg、开放结果、算术/位运算、表访问、控制流和执行资源预算。
- 添加循环捕获 cell、赋值左值求值顺序、多层闭包、递归、隐式 `self`、整数边缘语义和 IR 伪造输入测试。
