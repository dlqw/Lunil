# Lunil 0.12.0-alpha.13

## Added

- Added canonical, signed required target labels to patch manifests and `lunil patch inspect`.
- Added case-sensitive exact-match target-label admission to `LuaPatchAcceptancePolicy` for binding
  patches to deployment environments, regions, shards, platforms, or rollout rings.

## Security

- A patch whose signed target selector does not match the local deployment target is rejected before
  a durable replay reservation is consumed. Target labels do not discover targets, grant
  capabilities, or expand runtime permissions.
