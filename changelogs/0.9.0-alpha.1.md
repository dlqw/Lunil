# Lunil 0.9.0-alpha.1

## Unified numeric specialization

- Tier 2 function-entry regions and loop-entry OSR now share the same verified numeric analysis,
  budget planning, emission, and canonical deoptimization behavior.
- A loop that does not satisfy the shared proof continues through the managed execution path.
- Exact 64-bit accounting, debug and GC guards, invalidation, NativeAOT interpreter fallback, and
  Lua 5.4 semantics remain unchanged.

## Documentation

- Updated the installation, embedding, compatibility, architecture, and performance references.
- Added deterministic generation checks for the published performance charts.
