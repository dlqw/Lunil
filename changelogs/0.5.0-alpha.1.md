# 0.5.0-alpha.1

## Added

- 新增 Windows、Linux、macOS 的 x64/Arm64 六目标构建产物与可复现 PowerShell 打包脚本。
- 新增三平台 CI、格式检查、测试、NuGet 与符号包构建，以及标签驱动的 GitHub Release/CD 流水线。
- 新增受保护的 `main` 分支工作流、分支命名规范、CODEOWNERS、PR 模板与 Dependabot 配置。
- 新增 SemVer 2.0 版本策略，支持 `alpha.N`、`beta.N`、`rc.N` 和稳定版自动晋级与校验。
- 新增项目 SVG Logo、英文默认 README、简体中文 README、语言索引和完整文档导航。
- 新增 MIT 开源许可证，并将许可证元数据写入所有可发布 NuGet package。

## Changed

- 项目正式更名为 Lunil；解决方案、项目、程序集、命名空间、测试和 benchmark 标识统一迁移到 `Lunil.*`。
- NuGet 包、二进制 bundle、更新日志和 Git 标签统一使用同一个完整 SemVer 版本。
- 本地编译器缓存目录从 `.luac-cache` 更名为 `.lunil-cache`，避免与 Lua bytecode 的 `.luac` 扩展名混淆。
- 增加公开仓库边界校验，确保本地规划与 agent 工作文件不会进入 Git 跟踪或正式发布。

## Breaking changes

- 所有原 `Luac.*` .NET 命名空间和程序集名称更改为 `Lunil.*`；现有使用方需要更新项目引用与 `using` 指令。
