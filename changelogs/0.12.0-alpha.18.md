# Lunil 0.12.0-alpha.18

## Added

- Added patch-generation fencing for module-owned Lua coroutines, including suspended native
  `Yielded` and `CallLua` continuations.
- Added per-thread generation admission state and active, pending, quiesced, and stale native
  continuation gauges on `LuaHost`.

## Reliability

- Old module coroutines now fail closed after publication unless a signed runtime-owned
  `Coroutine`/`Continue` migration rule transfers them to the candidate generation. Transferred
  activations keep their old native descriptor, invocation state, callback frames, and Lua function
  versions; commit and rollout rollback restore the previous generation and reject candidates.
