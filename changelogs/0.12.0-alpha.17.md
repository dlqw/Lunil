# Lunil 0.12.0-alpha.17

## Added

- Added patch-generation fencing for module-owned `LuaClrTask` wrappers.
- Added task activity state and active, pending, quiesced, and stale task gauges.

## Reliability

- `clr.await` now rejects old-generation task results and blocks unpublished candidates outside
  their owning loader before they can enter another generation. Commit and rollout rollback restore
  the previous task generation without cancelling the underlying host operation.
