# Lunil 0.12.0-alpha.5

Lunil 0.12.0-alpha.5 adds first-class call sites, reference indexes, and immutable call-graph
projections for static-analysis hosts.

## Code intelligence

- Added typed call sites for direct, member, method, callable, dynamic, and unresolved calls.
- Added per-compilation call graphs with containing and target function IDs while retaining calls
  that have no static target.
- Added symbol and global-name reference queries to `LuaSemanticModel`.
- Added workspace-wide stable reference results and call graphs with module/source identities,
  stable function keys, and conservative `require` module/export targets.
- Shadowed `require`, dynamic requests, and reassigned module aliases remain explicit rather than
  being reported as resolved module edges.
