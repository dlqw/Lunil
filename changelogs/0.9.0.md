# Lunil 0.9.0

Lunil 0.9.0 is the stable release for .NET 10 and Lua 5.4.8.

## Highlights

- The default Auto JIT handles stable string-number and string-float concatenation, string length,
  and dense string-array writes while preserving Lua operand order, table guards, write barriers,
  instruction budgets, binary-string semantics, and canonical restart behavior.
- The existing dense `table.concat` implementation remains unchanged.
- The reference interpreter, adaptive JIT, NativeAOT, trimming, single-file, and ReadyToRun
  deployment modes remain available.

## Performance

- Auto JIT geometric mean: `1.688x` PUC Lua 5.4.8 and `24.089x` MoonSharp 2.0.0.
- Lowest per-RID Auto geometric mean: `1.103x` PUC Lua 5.4.8.
- `control_flow`: `2.101x`; `string_build`: `2.164x`.
- Results cover `win-x64`, `win-arm64`, `linux-x64`, `linux-arm64`, `osx-x64`, and `osx-arm64`.

The versioned dataset is available in the
[0.9.0 performance data](../benchmarks/results/0.9.0-performance.json) and
[performance documentation](../docs/performance.md).

## Compatibility

Native Lua C modules remain unsupported because Lunil does not expose the Lua C ABI.
