# Lunil 0.9.0-alpha.5

## String construction

- Improved short string and integer/float concatenation while preserving Lua GC visibility and
  binary-string semantics.
- Improved the existing `table.concat` write path without changing quota error order or native
  continuation behavior.
- Kept stable string operations, string length, and dense string-array writes in the shared guarded
  execution path.
- Added focused string, dense-array, and `table.concat` workloads for reproducible comparisons.

## Performance

- Improved the `string_build` workload to `2.164x` PUC Lua 5.4.8 in the published Alpha 5 dataset.
- Preserved exact instruction accounting, write barriers, canonical restart behavior, and the
  managed fallback path.
