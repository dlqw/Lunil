# Lunil 0.9.0-alpha.5

## 字符串构建与内存调度

- 优化短字符串与整数拼接的有界缓存，减少重复转换和临时分配，同时继续遵循 Lua GC
  的对象存活状态与二进制字符串语义。
- 调整增量 GC 的自动周期阈值和默认步进预算，使高分配 workload 的回收节奏更平滑。
- 优化 `table.concat` 的双片段写入路径，并保持配额错误顺序和可恢复 native continuation 行为。

## 验证

- 运行时 GC、弱表、finalizer、值语义、协程和嵌套 native callback 回归通过。
- 标准库回归通过；跨运行时性能报告和复现方式见[性能文档](../docs/performance.md)。

Alpha 版本仍处于功能冻结前阶段，API 和后端行为可能继续变化。
