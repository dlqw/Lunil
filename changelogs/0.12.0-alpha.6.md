# Lunil 0.12.0-alpha.6

## Added

- Added a runnable static-analysis embedding sample that exercises the compiler, EmmyLua/LuaLS
  annotations, semantic symbols and references, inferred types, CFGs, call graphs, diagnostics,
  and analysis budgets.
- Added an end-to-end workspace example for cyclic modules, fixed-point analysis, content-addressed
  cache reuse, export-driven invalidation, stable module/source identities, and workspace indexes.
- Added English and Simplified Chinese embedding tutorials covering UTF-8 byte spans, UTF-16 editor
  locations, snapshot-local IDs, stable keys, resolver naming, concurrency, disposal, and budgets.
