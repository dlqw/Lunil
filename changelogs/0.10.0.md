# Lunil 0.10.0

## Stable promotion

- Promote the accepted `0.10.0-rc.1` product to stable without product-code, public-API-shape,
  package-scope, canonical-IR, adapter, or evidence-schema changes.
- Freeze the 13-package `0.10.0` public API under the exact compatibility policy; patch releases on
  the `0.10.x` line accept backward-compatible fixes only.

## Lua 5.1–5.5 compatibility

- Added explicit Lua 5.1, 5.2, 5.3, 5.4, and 5.5 language profiles without implicit version
  fallback; Lua 5.4 remains the default when no version is selected.
- Added independent generated instruction codecs and binary-chunk adapters for all five PUC Lua
  formats, including Lua 5.5 varint/debug/register-layout boundaries.
- Completed GC scheduling and finalizer error propagation at the Lua 5.3 boundary, while keeping
  Lua 5.4 warning-and-continue behavior.
- Added Lua 5.5 named varargs and `global` declarations with canonical IR support and diagnostics.

## Standard-library and verification work

- Generated version capability profiles now select rawlen/unpack/loadstring/module, table/string/
  math/debug/package legacy surfaces, and Lua 5.5 `table.create` without leaking APIs across
  versions.
- Added five-version standard-library/error surface and behavior matrix tests, including Lua 5.3
  finalizer error propagation and Lua 5.1 legacy table iterators.
- PUC Lua differential gates cover Lua 5.1, 5.2, 5.3, 5.4, and 5.5 source/imported/produced
  chunks where the corresponding oracle is configured through `LUNIL_PUC_*` variables.
- Added a version-aware semantic cross-runtime comparison manifest for PUC Lua, NeoLua, UniLua,
  Luau, Wasmoon, and GopherLua.

## Release and compatibility

- Updated the reviewed 13-package public API baselines and package-consumer smoke gate.
- Release bundles and publish-mode gates remain defined for all six supported .NET RIDs.
- The accepted release candidate passed dual six-runner CI, full Release tests, formatting,
  conformance/differential, trimming, ReadyToRun, NativeAOT, package-consumer, and documentation
  gates, then published six RID bundles and 13 package/symbol pairs with SHA-256 asset digests.
