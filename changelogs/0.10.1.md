# Lunil 0.10.1

## Lua 5.1 environment compatibility

- Inject the Lua 5.1 function environment into canonical upvalue zero for binary functions that
  execute `GETGLOBAL`/`SETGLOBAL`, including ancestors that create those closures.
- Preserve ordinary Lua 5.1 upvalue indexes and use the per-closure legacy environment fallback
  for functions without global opcodes.
- Main-chunk converters now identify only upvalue zero as the environment and preserve descriptor
  kinds for additional root upvalues.
