# 0.6.0-alpha.2

## 新增

- 新增 profile-guided CoreCLR Tier 2：owner-safe argument/site profile、分级 promotion/cache、constant fold、
  dead move、numeric specialization、boolean branch、bounded mono/poly table PIC、known-closure call guard、
  fixed result-window reuse、materialized DeoptMap、guard recovery/reprofile、Tier 2 event/counter 与全后端差分。
- 新增 Runtime ABI canonical-instruction observer，使 Tier 1、Tier 2 与 persisted CIL 能以精确 canonical PC
  采集 profile，同时避免 backend entry 与 emitted instruction 的重复计数。

## 验证

- Tier 2 加入 interpreter、executor、persisted CIL AOT 与 Tier 1 的全 canonical opcode differential corpus。
- 覆盖并发 promotion 去重、随机 guard failure、table PIC mono/poly/megamorphic、tiered LRU eviction、
  owner unload、动态 reprofile、instruction accounting 与 debug/hook fallback。
