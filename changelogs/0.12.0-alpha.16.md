# Lunil 0.12.0-alpha.16

## Added

- Added automatic patch-generation fencing for module-owned CLR delegates and event subscriptions.
- Added active, pending, quiesced, and stale callback counters plus subscription activity evidence.

## Reliability

- Candidate callbacks remain pending until atomic publication. Commit, barrier, migration, and
  health-check rollback restore previous subscriptions and reject candidate callbacks; successful
  publication detaches superseded event handlers and rejects old delegates before Lua entry.
