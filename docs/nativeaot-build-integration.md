# .NET NativeAOT and trimming

The NativeAOT and trimming integration guide is maintained as a standalone Simplified Chinese
document: [.NET NativeAOT and trimming (简体中文)](nativeaot-build-integration.zh-CN.md).

Lunil supports source and chunk compilation plus interpreter execution when dynamic code is
unavailable. The guide lists supported publish modes, fallback behavior, and the six release RIDs.
